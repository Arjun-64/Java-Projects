# Aadhar Registration Form Application

The application collects user details through a registration form and stores the data securely in a database.

## Project Overview
The goal of this project is to design a simple and efficient Aadhar registration system that demonstrates form handling, validation, and backend integration using Spring Boot.

## Features
- User registration form for Aadhar details
- Field validation (name, age, address, mobile number, etc.)
- Data storage using database
- Spring Boot MVC architecture
- Clean and user-friendly UI

## Tech Stack

- CSS

## Modules
- Aadhar Registration Form
- User Details Processing
- Database Integration
